Welcome to MediaWiki.org
MediaWiki is a free software open source wiki package written in PHP, originally for use on Wikipedia. It is now used by several other projects of the non-profit Wikimedia Foundation and by many other wikis, including this website, the home of MediaWiki.

Use the links below to explore the site contents. You'll find some content translated into other languages, but the primary documentation language is English.

For general questions about MediaWiki see the communication page. If you have a question about this wiki, please visit the forum.
